## 三、发现的问题

### 1. 需要手动配置的项目（重要）

#### 环境 ID
- **位置**: `miniprogram/app.js:7`
- **当前值**: `env: 'your-env-id'`
- **需改为**: 实际的微信云开发环境 ID

**解决步骤**:
1. 登录微信云开发控制台
2. 复制环境 ID（格式类似：cloud1-xxx）
3. 替换 app.js 中的 `your-env-id`

### 2. 数据库权限配置

需要在微信云开发控制台配置 6 个集合的权限：

```
users - 所有用户可读，仅创建者可写
services - 所有用户可读，仅管理员可写
orders - 仅创建者可读写，管理员可读写
members - 仅创建者可读写，管理员可读写
schedules - 所有用户可读，仅管理员可写
reviews - 所有用户可读，仅创建者可写
```

### 3. 测试账号准备

- 需要在 `users` 集合手动创建老板账号：
  ```json
  {
    "openid": "xxx",
    "nickname": "测试老板",
    "role": "boss"
  }
  ```
