# 中医按摩推拿店预约小程序

## 项目简介

基于微信小程序原生开发 + 微信云开发的中医按摩推拿预约系统，支持顾客端预约和老板端管理功能。

## 技术栈

- **前端**：微信小程序原生开发
- **后端**：微信云开发（Serverless）
  - 云函数：业务逻辑处理
  - 云数据库：数据存储
  - 云存储：图片资源存储

## 功能模块

### 顾客端
- 首页：展示热门服务
- 服务列表：查看所有服务项目
- 服务详情：服务信息与在线预约
- 我的订单：预约订单管理
- 个人中心：个人信息与会员中心

### 老板端
- 数据看板：经营数据统计
- 服务管理：服务项目增删改
- 订单管理：订单状态管理
- 统计分析：数据报表

## 项目结构

```
miniprogram/
├── app.js                 # 小程序入口
├── app.json               # 小程序配置
├── pages/
│   ├── customer/          # 顾客端页面
│   └── boss/              # 老板端页面
├── components/            # 公共组件
├── cloud/                 # 云函数
└── utils/                 # 工具函数
```
